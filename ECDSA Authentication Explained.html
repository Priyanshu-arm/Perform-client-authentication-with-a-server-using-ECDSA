What is the Game?

The goal is:
Make sure the server knows it's really the correct client talking to it.

This is done using a "magic signature trick" (called ECDSA) where:

  • The Client has a secret key (like a password).

  • The Server knows the client's public key (like knowing your friend's face).

  • They talk using the internet (TCP/IP using Boost Asio).

  • All the messages are packaged neatly (using nanopb, a lightweight Protocol Buffers library).

Step-by-Step Explanation of the Game

Setup Phase:

  • Use Boost Asio to create a basic client and server that can send and receive messages.

Authentication Game Round 1

Step 1: Client starts the handshake

  • Client hashes its serial ID (think: turning name into a code using SHA256).

  • It signs the hash using its private key (the magic trick).

  • It sends this signature and serial ID to the server.

Round 2: Server Checks It

Step 2: Server verifies

  • Server hashes the received serial ID (same way).

  • It uses the public key to verify the signature.

  • If it matches, client is probably real!

Round 3: Random Challenge

Step 3: Server sends challenge

  • Server creates a random 32-byte number .

  • Sends it to the client.

Round 4: Client Responds Again

Step 4: Client signs challenge

  • Client hashes the random number.

  • Signs it with its private key.

  • Sends the signature back.

Final Round: Server Verifies

Step 5: Server checks again

  • Server hashes the random number.

  • Verifies the new signature with the public key.

  • If it's valid, it says:

    " Client Verified! Welcome, [serial ID]!"

Step 2: Generate ECDSA Keypair (secp256k1)

Create a new file called keygen.cpp:

cpp
#include <stdio.h>
#include <stdint.h>
#include <string.h>
#include "ecdsa.h"
#include "secp256k1.h"
#include "bignum.h"

int main() {
    // 32-byte private key
    const char* private_key_hex = "1c3d5e7a9b2d4f6a1e3c5d7f9b2a4c6e8d0f123456789abcdef1234567890abcd";
    uint8_t private_key[32];
    hex_decode(private_key, 32, private_key_hex);

    // Calculate public key
    uint8_t public_key[65];
    ecdsa_get_public_key65(&secp256k1, private_key, public_key);

    printf("Private Key: %s\n", private_key_hex);
    printf("Public Key: ");
    for (int i = 0; i < 65; i++) {
        printf("%02x", public_key[i]);
    }
    printf("\n");

    return 0;
}

Step 3: Client Signs Serial ID

cpp
#include <iostream>
#include <string>
#include "sha2.h"
#include "ecdsa.h"
#include "secp256k1.h"
#include "bignum.h"

int main() {
    const std::string serial_id = "CLIENT12345678";

    // Hash serial ID using SHA256
    uint8_t hash[32];
    sha256_Raw((const uint8_t*)serial_id.c_str(), serial_id.length(), hash);

    // Private key
    const char* private_key_hex = "1c3d5e7a9b2d4f6a1e3c5d7f9b2a4c6e8d0f123456789abcdef1234567890abcd";
    uint8_t private_key[32];
    hex_decode(private_key, 32, private_key_hex);

    // Sign hash
    uint8_t signature[64];
    ecdsa_sign(&secp256k1, private_key, hash, signature, NULL);

    std::cout << "Signature: ";
    for (int i = 0; i < 64; ++i) {
        printf("%02x", signature[i]);
    }
    std::cout << std::endl;

    return 0;
}

Step 4: Server Verifies Signature

On server side, use:

cpp
bool valid = ecdsa_verify(&secp256k1, public_key, hash, signature);
if (valid) {
    std::cout << " Signature is valid.\n";
} else {
    std::cout << " Signature is NOT valid.\n";
}

  • Click New File server.cpp

STEP 3: Paste the code

Paste this into server.cpp:

server.cpp

cpp
#include <iostream>
#include <boost/asio.hpp>

using boost::asio::ip::tcp;

int main() {
    try {
        boost::asio::io_context io_context;

        tcp::acceptor acceptor(io_context, tcp::endpoint(tcp::v4(), 1234));
        std::cout << "[Server] Listening on port 1234...\n";

        tcp::socket socket(io_context);
        acceptor.accept(socket);
        std::cout << "[Server] Client connected!\n";

        char data[1024] = {0};
        size_t length = socket.read_some(boost::asio::buffer(data));
        std::cout << "[Server] Received: " << std::string(data, length) << "\n";

    } catch (std::exception& e) {
        std::cerr << "Error: " << e.what() << "\n";
    }

    return 0;
}

Now paste this into client.cpp:

client.cpp

cpp
#include <iostream>
#include <boost/asio.hpp>

using boost::asio::ip::tcp;

int main() {
    try {
        boost::asio::io_context io_context;

        tcp::resolver resolver(io_context);
        auto endpoints = resolver.resolve("127.0.0.1", "1234");

        tcp::socket socket(io_context);
        boost::asio::connect(socket, endpoints);

        std::string message = "Hello, Server!";
        boost::asio::write(socket, boost::asio::buffer(message));

        std::cout << "[Client] Sent: " << message << "\n";

    } catch (std::exception& e) {
        std::cerr << "Error: " << e.what() << "\n";
    }

    return 0;
}
